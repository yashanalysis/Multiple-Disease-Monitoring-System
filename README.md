# Multiple-Disease-Monitoring-System
A web-based machine learning application that predicts multiple diseases such as Diabetes, Heart Disease, and Parkinson’s Disease based on user input parameters. The system helps users monitor health risks through an easy-to-use dashboard.
The Multiple Disease Monitoring and Prediction System is a healthcare-based web application developed using Python and Flask. It uses trained machine learning models to predict the likelihood of different diseases based on medical input data provided by users.

This project currently supports prediction for:

Diabetes Disease
Heart Disease
Parkinson’s Disease

Users can enter health-related values through simple forms and instantly receive prediction results. The project is designed to demonstrate how machine learning can be integrated into healthcare systems for early disease detection and monitoring.
